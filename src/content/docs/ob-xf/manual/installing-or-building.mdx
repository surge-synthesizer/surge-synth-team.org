# Installing OB-Xf

## Windows

On the Windows platform, OB-Xf installs optionally as:

- 64-bit VST3, with filename `OB-Xf.vst3`
- 64-bit CLAP plug-in instrument, with filename `OB-Xf.clap`
- Standalone executable, with filename `OB-Xf.exe`

The installer will install assets to the appropriate location for each format.
Windows builds are also available for arm64 and arm64ec, separately.


## macOS

On macOS, OB-Xf installs as a 64-bit Plug-in Instrument for the Audio Unit (AU), VST3, and CLAP plug-in interfaces,
and as a standalone application. It requires macOS 10.15 or later.

To install OB-Xf, run the packaged installer. You have the option to install AU `OB-Xf.component`, VST3,
and CLAP, along with factory patches and wavetables. The packaged installer installs OB-Xf for all users
of your system.


## Linux

On Linux, OB-Xf is available as a 64-bit VST3 CLAP and Standalone as a **deb** package, as a **zip**
file or as source.

The installation package on the **OB-Xf website** is a Debian package.

The distribution package is built on Ubuntu 22.04

A list of required packages is listed in the source and in the deb file.

## Standalone Version Options

If you are running OB-Xf as a standalone executable, options at the top-left:
- Configure MIDI and audio inputs and outputs
- Save and open state files for the synth
- Reset OB-Xf to the default state
