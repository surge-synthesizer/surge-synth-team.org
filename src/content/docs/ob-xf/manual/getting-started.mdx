# Getting Started

OB-Xf is accessible on Windows and macOS and works well with screen readers.

OB-Xf is a virtual synthesizer inspired by the venerable Oberheim OB-X. It is a continuation and modernatization of the last open source release of OB-Xd by 2DaT and later discoDSP. It is maintained by the Surge Synth Team, a group of volunteers.

This manual has the following main sections:

-   [Installing or building](/ob-xf/manual/installing-or-building/)
-   [Interface and using OB-Xf](/ob-xf/manual/user-interface-basics/)

To ask questions, report bugs or issues, or help develop OB-Xf, visit the [Surge Synth Team Discord server](https://discord.com/invite/spGANHw).

Most of the images and descriptions use the **Default** theme, and this manual uses macOS conventions (**Title Case**) for labels in the UI, and Windows keyboard labels.
