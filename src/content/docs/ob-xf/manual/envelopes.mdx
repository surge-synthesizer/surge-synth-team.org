# Filter and Amplifier Envelopes
OB-Xf features two ADSR envelopes per voice: one for the filter and one for the amplifier. Both envelopes have the same set of parameters:

## Basic Parameters
- **Attack**: This knob sets the time it takes for the envelope to rise from 0 to its maximum level after a note is pressed. The attack time can range from very fast (1 ms) to very slow (up to 60 seconds).
- **Decay**: This knob sets the time it takes for the envelope to fall from its maximum level to the sustain level after the attack phase is complete. The decay time can also range from very fast (1 ms) to very slow (up to 60 seconds).
- **Sustain**: This knob sets the level at which the envelope will hold after the decay phase, as long as the note is held. The sustain level can be set from 0% (no sustain) to 100% (full sustain).
- **Release**: This knob sets the time it takes for the envelope to fall from the sustain level back to 0 after the note is released. The release time can range from very fast (1 ms) to very slow (up to 60 seconds).

## Curve Sliders
Each envelope also features sliders that control the curvature of the Attack and Release stages. Moving the slider to the left creates a logarithmic curve (faster start), while moving it to the right produces a linear ramp.
