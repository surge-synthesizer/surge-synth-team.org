## Patch Organization

Each patch bank is separated in 16 patch groups, each containing 16 patches. The 16 buttons spanning from the center to the right of that bottom section can be used to select a group and a patch within that group. 

## Selecting a Patch or Patch Group
The yellow light corresponds to the currently selected group, while the red light corresponds to the currently selected patch.

To select a group instead of a patch with those buttons, click on the **Group** button first. Its yellow light will turn on, indicating that you are in group selection mode. Alternatively, you can also hold Ctrl/Cmd while clicking on one of the 16 buttons to select a group directly, or a patch when in group selection mode.

## Patch Menu, Cycling and Saving

- **Patch Menu**: Displays the current patch name and number. Click on it to open a dropdown menu to select patches directly.
- **Previous and Next Buttons**: These buttons allow you to cycle through patches sequentially.
- **Init**: Initializes the current patch to a basic saw wave sound.
- **Rand**: Randomizes all parameters of the current patch, which allows for unexpected results.
