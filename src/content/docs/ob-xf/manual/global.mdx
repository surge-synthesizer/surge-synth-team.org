# Global

The Global section contains different options that affect the entire synthesizer:


## Poly

This value sets the maximum number of voices that can be played simultaneously, from 1 to 32.


## Unison

Enables the layering of multiple voices per note. The number of voices is set by the [Voices](#voices) parameter.


## Voices

This value controls the number of voices played per note, from 1 to 8. 
- Only enabled when [Unison](#unison) is switched on.
- Higher values create a thicker sound by playing more voices simultaneously, but will reduce resulting polyphony.


## Detune

Adjusts the amount of pitch variation between voices from 0 to 100 cents.
- When [Unison](#unison) is disabled, increasing this control adds imperfection in the tuning between notes, better representing an OB-X that is not perfectly (or desperatly needs to be!) calibrated.
- When [Unison](#unison) is enabled, increasing this control creates a thicker, more detuned sound.


## HQ
Enables 2x oversampling at the oscillators and filters level. This improves audio quality at the cost of increased CPU usage.

## Env Legato

Sets which envelopes are not retriggered when playing legato (overlapping) notes:
- **Both (Both Envelopes)**: Neither the Amplifier nor the Filter envelope are retriggered.
- **Filter (Filter Envelope Only)**: Only the Filter envelope is not retriggered.
- **Amp (Ampplifier Envelope Only)**: Only the Amp envelope is not retriggered.
- **Retrig (Always Retrigger)**: Both envelopes are retriggered.


## Note Priority

Sets which note is prioritized when the maximum number of voices is reached:
- **Last**: The most recently played note is prioritized.
- **Low**: The lowest note is prioritized.
- **High**: The highest note is prioritized.



## Learn

Enables MIDI learn mode. When enabled, clicking on a parameter will assign it to the next received MIDI CC message.


## Menu

This button opens a menu with the following options:


### File
- **Initialize Patch**: Resets the synthesizer to its default state.
- **Import Patch**: Opens a file dialog to import a patch from a `.fxp` file.
- **Import Bank**: Opens a file dialog to import a bank of patches from a `.fxb` file.
- **Export Patch**: Opens a file dialog to export the current patch to a `.fxp` file.
- **Export Bank**: Opens a file dialog to export the current bank of patches to a `.fxb` file.
- **Copy Patch**: Copies the current patch to the clipboard.
- **Paste Patch**: Pastes a patch from the clipboard to the synthesizer.
- **Open User Directory**: Opens the user directory where patches, banks and themes are stored.


### Patches

This submenu contains a list of all groups and patches in the current bank. Selecting a patch from the list loads it into the synthesizer.


### Banks

This submenu contains a list of all available banks. Selecting a bank from the list loads it into the synthesizer.


### MIDI Mappings

#### Clear MIDI Mappings

Clears all MIDI mappings currently assigned to parameters.

### Themes

This submenu contains a list of all available Fatory and User themes. User themes can be added to the user directory (see [Open User Directory](#file) above).

### Zoom

This submenu contains options to adjust the zoom level of the interface, from 75% to 400%. Some zoom settings may not be available depending on your screen resolution.
